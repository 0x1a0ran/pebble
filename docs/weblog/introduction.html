What is Pebble weblog?

Pebble weblog is the popular server-side blogging tool written in Java. It's small, fast and feature-rich with an unrivalled ease of use. Blog content is stored as XML files on disk and served up dynamically, so there's no need to install a database. All maintenance and administration can be performed through your web browser, making Pebble ideal for anybody who is constantly on the move or doesn't have direct access to their host. A summary of the key features is as follows.

Blog Entries | Comments and TrackBacks | Web-based Adminstration | Newsfeeds | Navigation | Team, Project and Multi-user Blogs | i18n | Plugin APIs | Integration

Blog Entries
  • Post new blog entries through your web browser.
  • Post new blog entries using Blogger/MetaWeblog API compatible tools.
  • Organise your blog entries into one or more categories (hierarchical categories supported).
  • Tag your blog entries.
  • Add attachments to your blog entries for distrbuting files or podcasting.
  • Preview your blog entries before posting.
  • Save common content as templates for use later on.
  • Save blog entries as drafts.
  • Automatically ping websites via XML-RPC when new blog entries are added.
  • Publish static content alongside your blog entries.
Comments and TrackBacks
  • Readers can leave comments containing a safe subset of HTML.
  • Readers can reply to specific comments to create threads of discussion.
  • Automatically send e-mail notifications when new comments are left, with the ability to opt-out later.
  • Receive MovableType TrackBacks on your blog entries.
  • Disable comments and/or TrackBacks completely or on a per blog entry basis.
  • Approve new comments and TrackBacks (automatically or manually) before they appear on your blog.
  • Beat comment and TrackBack spam with the included user-configurable comment listeners.
  • Approve, reject or remove comments and TrackBacks in bulk or individually.
Web-based Adminstration
  • Perform all administration of your blog and plugins through your web browser.
  • Upload images and files to your blog.
  • Export your blog data as a ZIP file for backup.
  • Export your blog as RSS/RDF/Atom.
  • View referers and requests to your blog on a daily or monthly basis.
  • Export your log files in the standard Combined Log Format for complex analysis of your traffic.
  • Filter referer spam using regular expressions.
  • Switch and modify your look and feel using themes.
Newsfeeds
  • Publish your blog entries as RSS 2.0, RDF and Atom 1.0 feeds.
  • Publish your blog entries on a per category basis in the above feed formats.
Navigation
  • View each blog entry on a single page along with its comments and TrackBacks.
  • View all blog entries for a day.
  • View all blog entries for a month.
  • Navigate through your blog entries in a linear way or use the locale-aware calendar control with links to days containing entries and previous/next month.
  • View all entries added today using the permalinkable today page.
  • Search across blog entries, comments, TrackBacks and static content.
Team, Project and Multi-user Blogs
  • Let multiple people contribute to your team blog.
  • Host more than a single blog for a project, group or community.
  • Powerful aggregation features allow you to view, navigate and search across all multi-user blogs as if it was a single blog.
i18n
  • Publish content in international character sets such as UTF-8.
  • Publish content aligned with your timezone, regardless of where your host is based.
  • Configure your country/language to deliver content in a locale sensitive way.
  • Translations of public text provides a consistent user experience in a single language. The following languages are currently supported.

    • Chinese (Simplified)
    • Chinese (Traditional)
    • Danish
    • Dutch
    • English
    • Finnish
    • French
    • German
    • Hindi
    • Italian
    • Japanese
    • Norwegian
    • Portuguese
    • Spanish
    • Swedish
Plugin APIs
  • Blog entry decorators let you customize how blog entries are displayed.
  • Permalink providers let you customize the format of your permalinks.
  • Blog entry listeners let you write Java code that is called when blog entries are added and removed.
  • Comment listeners let you write Java code that is called when comments are added, removed, approved and rejected.
  • TrackBack listeners let you write Java code that is called when TrackBacks are added, removed, approved and rejected.
  • Blog listeners let you write Java code that is called when your blog starts up and shuts down.
Integration
  • Integrate Pebble with your existing security infrastructure (e.g. LDAP, Active Directory, RDBMS, etc) using the standard J2EE security mechanism to roll out access across the enterprise.
  • Publish build results with tools such as CruiseControl and Ant, taking advantage of the "build" theme to turn your blog into an extreme feedback device.
  • Use Pebble as the knowledge management hub for your project, integrating data from other blogs, automated builds, bug trackers and anything that can generate an RSS/Atom feed.

Download Pebble now and start blogging in a couple of minutes.

Minimum requirements
  1. Java 2 Platform, Standard Edition (J2SE) 1.4
  2. A JSP 1.2/Servlet 2.3 compatible web/application server such as Tomcat, Jetty or Resin, with access to setup new users and roles.
