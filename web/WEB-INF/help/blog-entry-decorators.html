Blog Entry Decorators

Blog entry decorators are a type of Pebble plugin that provides a way to change the way that blog entries are displayed in the HTML pages of your blog and the XML newsfeeds that are generated. The following pre-built decorators are included in the distribution.

net.sourceforge.pebble.plugin.decorator.HideUnapprovedBlogEntriesDecorator Hides blog entries that do not have a status of approved, unless you are logged in as a blog owner or contributor.
net.sourceforge.pebble.plugin.decorator.HideUnapprovedResponsesDecorator Hides comments and TrackBacks that do not have a status of approved, unless you are logged in as a blog owner or contributor.
net.sourceforge.pebble.plugin.decorator.HtmlDecorator Filters the HTML inside comments/TrackBacks to use only a limited safe subset and allows blog entries to be rendered as HTML.
net.sourceforge.pebble.plugin.decorator.RadeoxDecorator Renders content inside blog entry/static pages between <wiki>...</wiki> tags using the Radeox library.

Internal style wiki links (e.g. [aboutme]) link to the static page with the same name. If the static page doesn't exist, the link lets you create the page.
net.sourceforge.pebble.plugin.decorator.EscapeMarkupDecorator Escapes HTML markup between <escape>...</escape> tags (like this except that the tags are maintained and simply decorated out at runtime). This is useful where you would like to post large fragments of HTML, JSP, XML or Java code. In doing so, any < or > characters will be automatically escaped to &lt; and &gt; respectively.
net.sourceforge.pebble.plugin.decorator.RelativeUriDecorator Replaces relative URIs with absolute URLs so that they are rendered properly in browsers and newsreaders.

For example, a link to ./images/someimage.jpg would be transformed to http://www.yourdomain.com/blog/images/someimage.jpg.
net.sourceforge.pebble.plugin.decorator.NoFollowDecorator Provides support for the nofollow initiative by modifying all links in comments and TrackBacks.
net.sourceforge.pebble.plugin.decorator.BlogTagsDecorator Adds links from your blog entry back to your tags.
net.sourceforge.pebble.plugin.decorator.TechnoratiTagsDecorator Adds links from your blog entry back to tags on Technorati.
net.sourceforge.pebble.plugin.decorator.DisableResponseDecorator Disables comments and TrackBacks for the blog entry. This is useful if you are worried about spam when you don't have access to your blog.

Also available are net.sourceforge.pebble.plugin.decorator.DisableCommentsDecorator and net.sourceforge.pebble.plugin.decorator.DisableTrackBacksDecorator that disable comments and TrackBacks, respectively.

By default, Pebble will use the following blog entry decorators if none are configured. 

net.sourceforge.pebble.plugin.decorator.HideUnapprovedBlogEntriesDecorator
net.sourceforge.pebble.plugin.decorator.HideUnapprovedResponsesDecorator
net.sourceforge.pebble.plugin.decorator.HtmlDecorator
net.sourceforge.pebble.plugin.decorator.EscapeMarkupDecorator
net.sourceforge.pebble.plugin.decorator.RelativeUriDecorator
net.sourceforge.pebble.plugin.decorator.NoFollowDecorator
net.sourceforge.pebble.plugin.decorator.BlogTagsDecorator
To change this, simply modify the list of blog entry decorators on the plugins page.
