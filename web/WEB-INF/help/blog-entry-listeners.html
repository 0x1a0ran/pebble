Blog Entry Listeners

Blog entry listeners are a type of Pebble plugin that allow custom code to be called whenever a blog entry is added or removed. The following blog entry listeners are included with the Pebble distribution.

net.sourceforge.pebble.event.blogentry.EmailNotificationListener Sends a notification e-mail when a new blog entry is added or changed.

The list of recipients can be specified in the E-mail notifications section of the blog properties page.
net.sourceforge.pebble.event.blogentry.XmlRpcNotificationListener Sends an XML-RPC update notification ping when a blog entry is added.

The list of websites can be specified in the XML-RPC update notifications section of the blog properties page.
net.sourceforge.pebble.event.blogentry.MarkPendingListener Marks all new and changed blog entries with a status of pending so that they require approval from the blog owner before being published on the blog. Only the following subset of properties, when changed, will cause a blog entry to be marked as pending.
  • Title
  • Excerpt
  • Body
  • Categories
  • Original permalink
  • Attachment

To configure the blog entry listeners used by your blog, simply modify the list on the plugins page. By default, no blog entry listeners are used.
